# Feasible Falcon

Project to play around and practice my skills with Kotlin, Spring WebFlux, AWS Kinesis, and the Twitter v2 Stream API

## Run Locally

This project uses `Gradle` as build tool and `Java 11` as runtime environment.

```bash
  gradlew bootRun
```
